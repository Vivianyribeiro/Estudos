software -> conjunto de instrucoes/programas/procedimentos que governam/gerenciam as acoes de uma maquina programavel

produto de software -> é um componente de um sistema de software que tem documentacao e pode ser execultado, testado, modificado e estendido por qualquer pessoa (pacote)

sistema de software -> é um software que - envolve varios programas interagindo entre si - programs  projetos para funcionarem integrada e cooperativamente

**** categorias de software

SOFTWARE APLICATIVO -> realiza processamento de informacoes para usuarios finais
	software aplicativo para propositos especificos -> programas que aopiam aplicacoes especificas dos usuarios (sigaa)
 			/ 
	software aplicativo para propositos geral -> prgms q realizam trabalho de processamento para usuarios finais (processamento de texto, planilhas)

SOFTWARE DE SISTEMAS -> Gerencia e suporta as operacoes dos sistemas computacionais e redes
	software para GERENCIAMENTO de sistemas (ex- windows, linux)
			 /
	software oara DESENVOLVIMENTO de sistemas (EX - ide, vscode, eclipse, pycharm)

***** softwares sao licenciados - nao da pra comprar um software, se compra uma licensa para usar ele

caracteristicas de software : 

• é especificado e não construído
• é uma entidade lógica (conceitual)
• é intangível
• não se desgasta
• não pode ser visualizado
• é inerentemente complexo
• precisa ser compatível
• sempre sofre modificações

ciclo de vida do software: fase ou etapas que o software passa

-concepcao -> desenvolvimento -> implantacao -> manutencao -> interrupcao de uso.

concepcao -> identificacao de um problema a ser resolvido por um sistema de software (requisitos escopo ccomplexidade)

desenvolviemnto -> elaboracao de uma solucao computacional que vai satisfazer os requisitos do sistemas de software (analise desenho da solucao, implementacao e teste)

implantação -> colocação do sistema em utilizacao 

manutencao -> aplicação de esforcos pra manter o sistema em operação (manutencao corretiva, adaptativa, preventiva, evolutiva)

interrupcao de uso -> deixa de ser usado quando a relacao entre o custo de sua manutencaoe o beneficio do seu uso se torna desfavoravel, por obsolencia tecnologica e mudancas extremas de ambiente de negocios tbm.
